Getting Started

This dashboard reads data from the .mission-control/ directory.

For Local Development:

  1. Start a local server: python -m http.server 8000
  2. Open http://localhost:8000/dashboard/
  3. Data will be loaded from the repository

For GitHub Pages:

  1. Enable GitHub Pages in repository settings
  2. Tasks will be loaded via GitHub API
